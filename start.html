Step 1

Skin is distributed as a npm module skin-ebay. If skin is not already installed in your project, install it using the below command 

npm install skin-ebay --save

Step 2

Include the required skin modules in optimizer.json and you are good to go 

{
    "dependencies": [
        "skin-ebay/core",
        "skin-ebay/button",
        "module.less",
        "require: ./widget"
    ]
}
