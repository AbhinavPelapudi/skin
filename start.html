Skin 2.0
  • Modules
  • Getting Started

Step 1

Skin is distributed as an NPM package. Install the package using the npm install command.

npm install skin-ebay --save

Skin 2.0 does not follow strict semantic versioning, it has the following convention instead:

  • PATCH version when we make backwards-compatible bug fixes AND when we add functionality in a backwards-compatible manner.
  • MINOR version when we make incompatible API changes.
  • MAJOR version when we make radical architectural changes (i.e this will be the next version of Skin, Skin3).

To avoid receiving breaking changes, please ensure that sure your package.json dependency is configured to receive patch updates only.

{
    "dependencies": {
        "skin-ebay": "~2.2.0"
    }
}

Step 2

Include the desired Skin modules in your browser.json and you are good to go.

{
    "dependencies": [
        "skin-ebay/core",
        "skin-ebay/button"
    ]
}

Skin 2.0 includes the following modules.

  • skin-ebay/button
  • skin-ebay/card
  • skin-ebay/combo
  • skin-ebay/core
  • skin-ebay/dialog
  • skin-ebay/form
  • skin-ebay/icon
  • skin-ebay/less
  • skin-ebay/menu
  • skin-ebay/tab
